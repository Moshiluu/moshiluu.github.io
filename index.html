Completed Projects

Planet Defender

Planet Defender was my first ever project using Unity. This was an introductory project to an online course that I am taking called GameDevRocket. It is a short, but sweet game with a simple premise: protect your planet from incoming asteriods.

Climb to Riches

Climb to Riches is the second game in the GameDevRocket course. In this game, you work up from almost nothing. You build houses that generate you income in hopes that one day, you can build the house of your dreams.


Current Projects

Winter MelonJam!

The MelonJam is a game jam (a timed game creation competiton) that will test my skills as the main programmer for my team. We will have 72 hours to produce a game based on the given prompt. This will test not only my programming skills, but also show off my communication and teamwork ability. The MelonJam runs from December 28th, 2023 to Demeber 30th, 2023.

Global Game Jam!

The Global Game Jam is a game jam (a timed game creation competition) hosted from all over the world. On the local level, certain places will host a game jam site which allows people like me to get together and create a game for the competition! The Global Game Jam will be a week long event running from January 22th, 2024 to January 28, 2024.
